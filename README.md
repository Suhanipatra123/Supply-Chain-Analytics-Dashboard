# 📦 Supply Chain Analytics Dashboard

## 📌 Project Overview
The **Supply Chain Analytics Dashboard** is an interactive Power BI project designed to monitor and analyze inventory, supplier performance, delivery operations, and overall supply chain efficiency.

This project helps businesses track:
- Inventory availability
- Delivery delays
- Supplier reliability
- Profit & loss distribution
- Order trends and operational risks

The dashboard transforms raw supply chain data into meaningful business insights for better decision-making and operational improvement.



# 🚀 Project Objectives

✅ Monitor inventory stock across locations  
✅ Analyze delivery performance and delays  
✅ Evaluate supplier reliability and quality  
✅ Track order trends and operational KPIs  
✅ Identify risk areas in the supply chain  
✅ Improve business visibility using interactive dashboards  



# 🛠️ Tools & Technologies Used

- **Power BI**
- **Power Query**
- **DAX**

- **Data Cleaning & Data Modeling**
